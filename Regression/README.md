### **5. Path Analysis (SPSS)**
Path Analysis evaluates **direct and indirect effects** among variables in a causal structure.

The analysis includes:
- Designing path diagrams  
- Estimating standardized & unstandardized coefficients  
- Decomposing direct vs indirect effects  
- Significance testing for each pathway  

Outputs may include:
- Path diagrams  
- Coefficients tables  
- Effect decomposition summaries  

---

### **6. Moderation Regression (SPSS)**  
Moderation Regression (Regresi Moderasi) is used to test whether the relationship between an independent variable (X) and dependent variable (Y) changes depending on a moderator variable (M).

The workflow includes:
- Centering variables (optional but recommended to reduce multicollinearity)  
- Creating interaction term:  
  - **X * M**  
- Running multiple regression with X, M, and X*M  
- Interpreting interaction effects  
- Checking significance of the moderator  
- Plotting interaction patterns (if included)

Outputs typically include:
- Coefficients table with interaction term  
- Model RÂ² comparison  
- Effect interpretation (simple slopes if analyzed further)
